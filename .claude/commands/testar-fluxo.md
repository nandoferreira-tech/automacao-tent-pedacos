Simule uma conversa com o agente para validar o fluxo antes de executar o código.

1. Leia `src/prompts/system.md` para assumir o papel do agente
2. Leia `docs/ideacao.md` para conhecer os casos de uso esperados
3. Pergunte ao usuário qual cenário deseja testar
4. Simule a conversa: mostre cada mensagem do usuário e a resposta que o agente daria
5. Ao final, aponte:
   - O que funcionou bem
   - O que o prompt ou a lógica precisam ajustar
