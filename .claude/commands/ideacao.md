Conduza uma sessão de ideação para o agente WhatsApp deste projeto.

1. Leia o arquivo `docs/ideacao.md` para entender o estado atual
2. Faça perguntas para levantar:
   - Objetivo principal do agente
   - Quem são os usuários (público-alvo)
   - Quais mensagens o agente deve saber responder
   - Quais ações o agente deve executar (consultar API, salvar dados, etc.)
   - Se precisa de memória de conversas
3. Ao final, atualize `docs/ideacao.md` com as respostas coletadas
